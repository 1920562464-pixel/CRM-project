# 分角色仪表盘系统

## 概述

全新的分角色仪表盘系统已上线！系统会根据当前登录角色自动展示相应的功能模块、统计数据和待办事项。

## 支持的角色

### 1. 管理员 (admin)
**统计数据：**
- 系统用户总数
- 本月收入
- 待办事项
- 系统状态

**功能模块：**
- 客户管理
- 库存管理
- 资产管理
- 财务中心
- 报销管理
- 系统管理

**快捷操作：**
- 用户管理
- 系统设置
- 数据报表
- 审核中心

## 功能特性

### 1. 智能问候
根据当前时间段自动显示：
- 凌晨好 (0:00 - 6:00)
- 早上好 (6:00 - 9:00)
- 上午好 (9:00 - 12:00)
- 中午好 (12:00 - 14:00)
- 下午好 (14:00 - 18:00)
- 晚上好 (18:00 - 22:00)
- 夜深了 (22:00 - 24:00)

### 2. 待办任务管理
- 查看全部任务
- 搜索任务
- 筛选状态（全部/待完成/已完成）
- 标记完成
- 删除任务
- 优先级标识（紧急）

### 3. 统计卡片
- 显示关键指标
- 趋势变化指示
- 点击交互

### 4. 快捷操作
- 角色专属快捷入口
- 一键跳转

### 5. 最新动态
- 实时更新
- 角色相关

### 6. 数据概览
- 角色专属数据面板
- 进度追踪

## 角色切换

在系统顶部导航栏可以切换不同角色，仪表盘会自动根据当前角色展示相应内容。

---

## 技术实现

- **Vue 3** Composition API
- **TypeScript** 类型支持
- **markRaw** 优化组件渲染
- **响应式计算** 动态数据更新
- **Vue Router** 路由导航

---

## 注意事项

1. 所有图标组件使用 `markRaw` 包装，避免 Vue 的响应式代理导致性能问题
2. 角色配置由 `useRole` composable 提供
3. 数据可后续替换为真实 API 调用
