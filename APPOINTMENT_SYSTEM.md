# 医生上线服务预约系统

## 功能概述

教练/顾问可以通过仪表盘预约医生的上线服务，医生可以接受/拒绝预约，服务完成后自动添加200元到财务中心薪酬管理。

## 核心功能

### 1. 教练/顾问功能

#### 预约服务
- **入口**: 仪表盘 → 快捷操作 → "预约服务" 按钮
- **预约信息**:
  - 服务类型（在线问诊/处方审核/随访指导/紧急咨询）
  - 预约日期和时间
  - 选择医生（显示在线状态）
  - 服务时长（15-120分钟）
  - 客户姓名
  - 问题描述
  - 证明材料上传（支持拖拽，最大10MB）

#### 我的预约
- **位置**: 仪表盘右侧栏（教练角色时显示）
- **显示内容**:
  - 最近5条预约记录
  - 预约状态（待确认/已接受/已拒绝/已完成）
  - 服务详情
  - 待确认数量徽章

### 2. 医生功能

#### 服务预约管理
- **位置**: 仪表盘右侧栏（医生角色时显示）
- **功能**:
  - 查看分配给自己的预约请求
  - 接受/拒绝预约
  - 完成服务
  - 实时状态更新

#### 预约操作流程

1. **待接受状态** (pending)
   - 显示预约详细信息
   - 两个操作按钮: "接受" 和 "拒绝"
   - 黄色背景高亮

2. **已接受状态** (accepted)
   - 显示"已完成"按钮
   - 绿色背景
   - 提醒医生按时提供服务

3. **服务完成**
   - 点击"完成服务"按钮
   - 自动添加200元记录到财务中心
   - 显示确认消息

### 3. 财务集成

服务完成后，系统自动创建薪酬记录：

```typescript
{
  id: 'timestamp',
  type: 'online-service',
  amount: 200,
  description: '上线服务 - [客户名] - [服务类型]',
  date: 'ISO时间戳',
  appointmentId: '预约ID'
}
```

记录保存在 `localStorage` 的 `salary_records` 键下。

## 数据流程

```
教练点击"预约服务"
    ↓
打开 AppointmentModal
    ↓
填写预约信息 + 上传证明材料
    ↓
提交预约 → appointments 数组
    ↓
医生仪表盘显示待处理预约
    ↓
医生接受 → status: 'accepted'
    ↓
医生完成服务
    ↓
自动添加财务记录 → localStorage
```

## 状态管理

### 预约状态枚举

| 状态 | 英文 | 颜色 | 说明 |
|------|------|------|------|
| 待接受 | pending | 琥珀色 | 等待医生确认 |
| 已接受 | accepted | 绿色 | 医生已接受，等待服务 |
| 已拒绝 | rejected | 红色 | 医生拒绝该预约 |
| 已完成 | completed | 蓝色 | 服务已完成，已计费 |

### 服务类型映射

| 值 | 显示名称 |
|----|----------|
| consultation | 在线问诊 |
| prescription | 处方审核 |
| follow-up | 随访指导 |
| emergency | 紧急咨询 |

## UI 组件

### AppointmentModal 组件

**Props:**
- `showModal: boolean` - 控制模态框显示

**Emits:**
- `close` - 关闭模态框
- `submit: [appointment]` - 提交预约

**表单验证:**
- 服务类型（必填）
- 预约日期（必填，不能早于今天）
- 预约时间（必填）
- 医生选择（必填）
- 客户姓名（必填）
- 证明材料（必填，至少一个文件）

**文件上传特性:**
- 支持拖拽上传
- 支持多文件选择
- 文件大小验证（最大10MB）
- 文件类型限制（图片、PDF、Word）
- 预览已上传文件列表
- 可删除已上传文件

## 使用示例

### 教练预约服务

1. 切换到"教练"角色
2. 点击"快捷操作"中的"预约服务"按钮
3. 填写预约信息:
   - 选择服务类型: "在线问诊"
   - 选择日期: 明天
   - 选择时间: "09:00"
   - 选择医生: "李医生"
   - 客户姓名: "张建国"
   - 问题描述: "客户咨询血糖控制方案"
   - 上传证明材料（聊天记录截图等）
4. 点击"提交预约"
5. 查看右侧栏"我的预约"确认提交成功

### 医生处理预约

1. 切换到"医生"角色
2. 查看右侧栏"服务预约"
3. 点击预约卡片查看详情
4. 点击"接受"按钮确认预约
5. 服务完成后点击"完成服务"
6. 系统自动添加200元到财务中心

## 技术栈

- **框架**: Vue 3 Composition API
- **路由**: Vue Router 4
- **UI库**: Tailwind CSS
- **图标**: Lucide Vue Next
- **状态管理**: Ref + Computed
- **数据持久化**: localStorage

## 注意事项

1. **文件上传**: 当前仅前端实现，需要后端API支持
2. **医生认证**: 当前医生ID硬编码，需要与实际用户系统对接
3. **财务集成**: 当前使用localStorage模拟，需要连接真实财务系统
