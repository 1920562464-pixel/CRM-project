# 惯能健康 CRM - 系统架构图

## 整体架构

```
┌──────────────────────────────────────────────────────────────────┐
│                         用户界面层 (Views)                        │
├──────────────────────────────────────────────────────────────────┤
│  工作台  │  客户管理  │  财务管理  │  结算中心  │  ERP  │  智能辅助 │
│  ┌────┐  ┌─────────┐ ┌─────────┐ ┌──────────┐ ┌────┐ ┌────────┐ │
│  │仪表盘│  │客情监控 │ │财务概览 │ │教练结算  │ │库存│ │AI助手  │ │
│  └────┘  │客户列表 │ │收入管理 │ │医生结算  │ │资产│ │知识库  │ │
│          │潜客管理 │ │应收应付 │ │佣金结算  │ │    │ │预警中心│ │
│          └─────────┘ └─────────┘ └──────────┘ └────┘ └────────┘ │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│                       业务逻辑层 (Composables)                    │
├──────────────────────────────────────────────────────────────────┤
│  useRole  │ useTheme │ useToast │ useSalary │ useCommission     │
│  useFinancialData │ useRiskPrediction │ useRichBiscuit          │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│                         组件层 (Components)                       │
├──────────────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌────────────┐  ┌──────────────┐                │
│  │ shared/  │  │settlement/ │  │ knowledge/   │                │
│  │ Modal    │  │结算组件    │  │文件管理      │                │
│  │ Toast    │  │薪酬组件    │  │文件夹管理    │                │
│  │ DatePicker│ │佣金组件    │  │              │                │
│  └──────────┘  └────────────┘  └──────────────┘                │
│  ┌────────────────────────────────────────────┐                 │
│  │ 业务组件 (RoleBasedDashboard, Commission,  │                 │
│  │           RiskPredictionPanel, etc.)       │                 │
│  └────────────────────────────────────────────┘                 │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│                         数据层 (Data Layer)                       │
├──────────────────────────────────────────────────────────────────┤
│  types/        # TypeScript 类型定义                              │
│  data/         # 静态数据和 Mock 数据                              │
│  utils/        # 工具函数                                         │
└──────────────────────────────────────────────────────────────────┘
```

## 技术栈

### 前端框架
```
Vue 3.5.28          # 渐进式 JavaScript 框架
  ├─ Composition API
  ├─ <script setup>
  └─ TypeScript 5.9.3

Vue Router 4.6.4    # 官方路由管理器

Pinia 3.0.4         # Vue 官方状态管理库
```

### 构建工具
```
Vite 7.3.1          # 下一代前端构建工具
  ├─ 快速冷启动
  ├─ 即时模块热更新 (HMR)
  └─ 优化的生产构建
```

### UI 框架
```
Tailwind CSS 3.4.19 # 实用优先的 CSS 框架
  ├─ 响应式设计
  ├─ 主题定制
  └─ CSS 变量支持

Lucide Vue 0.564.0  # 现代化图标库
```

### 工具库
```
PapaParse 5.5.3     # CSV 解析
XLSX 0.18.5         # Excel 处理
```

## 关键文件说明

| 文件路径 | 说明 |
|---------|------|
| `src-vue/main.ts` | 应用入口，初始化 Vue 应用 |
| `src-vue/App.vue` | 根组件，包含侧边栏和主布局 |
| `src-vue/router/index.ts` | 路由配置，定义所有页面路由 |
| `src-vue/composables/*.ts` | 组合式函数，封装业务逻辑 |
| `src-vue/types/*.ts` | TypeScript 类型定义 |
| `src-vue/components/shared/` | 共享组件库 |
| `src-vue/views/*.vue` | 页面级组件 |
| `package.json` | 项目依赖配置 |
| `vite.config.vue.ts` | Vite 构建配置 |
| `tailwind.config.js` | Tailwind CSS 配置 |
